HEIC形式の画像をJPGに変換する方法

Feb 1, 2024 · 6 mins read
ブログツール

iphoneで撮影した画像を取り込んでブログに・・・と思ったところ、拡張子がHEICだったのでホームページビルダーに親しんでいた私としては困りました。

そもそもHEIC（High Efficiency Image File Format）とは、AppleがiPhoneやiPadで採用している画像形式のようで、 JPEGに比べてファイルサイズを小さく保ちながら高い画質を維持できるため、ストレージの節約に役立っているようです。 ただし、このままでは、すべてのプラットフォームやウェブサイトがHEICをサポートしているわけではないため、互換性の観点からJPGに変換する必要がある場合があります。

そこで今回はこのHEIC形式の画像をJPGに変換する方法を書き残しておきます。

MacでのHEICからJPGへの変換方法

Macユーザーであれば、画像のバッチ処理を行うためにImageMagickというツールを利用するのが便利です。 ImageMagickは、画像の変換、作成、編集を行うための強力なコマンドラインツールセットです。

ImageMagickのインストール

ImageMagickはHomebrewを通じて簡単にインストールできます。 Homebrewは、macOS用のパッケージマネージャーで、以下のコマンドでImageMagickをインストールできます： 

brew install imagemagick
このコマンドは、MacにImageMagickをダウンロードしてインストールします。

HEICからJPGへの変換 ImageMagickをインストールした後、変換したいHEICファイルがあるディレクトリに移動し、以下のコマンドを実行します： 

mogrify -format jpg *.HEIC
このコマンドは、指定されたディレクトリ内のすべてのHEICファイルをJPGに変換します。 mogrifyはImageMagickのコマンドの1つで、画像を直接変更できます。

HEIC形式について

余談ですが、HEIC形式は、Moving Picture Experts Group（MPEG）によって開発されたHigh Efficiency Image File Format (HEIF) の一部です。 HEICは「High Efficiency Image Coding」の略で、MPEG-H Part 12で定義されています。 HEIC形式は、JPEGよりも優れた圧縮を提供し、画像品質を維持しながらファイルサイズを小さくできます。 また、透明度情報や複数の画像を1つのファイルに格納するなど、追加の機能もサポートしています。

まとめ

iPhoneのHEIC画像をJPGに変換することは、とくにウェブ互換性や共有を考慮すると重要です。Macを使用している場合、ImageMagickのようなツールを利用すると、簡単にバッチ処理で画像を変換できます。これにより、効率的にファイル形式の変更が行え、作業の手間を大幅に削減できます。
